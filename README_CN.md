# APITool

简体中文 / [English](./README.md)

APITool 是一个可本地部署的、打通前后端的可视化的接口管理平台

## 使用技术

- React（Vite）：用于前端
- NestJS：用于后端
- PostgreSQL（主要数据库）
- Prisma ORM：可以轻松切换到任何其他关系型数据库，只需对代码进行少量修改
- Redis（用于缓存、会话存储和接口统计）
- SMTP 服务器（用于发送密码恢复邮件）
- GitHub/Google OAuth（用于快速认证用户）
- LinguiJS 和 Crowdin（用于翻译管理和本地化）
